-------------------------
R47 Firmware Update Guide
=========================


--------------------------------------
BEFORE YOU START - BACKUP YOUR DATA
--------------------------------------
Step 1: Create a State Backup
Access: g1 or use keys g 1
Choose ONE of these backup methods:

Method A (Minimum): g0, F3(SAVE)
→ Creates backup file in SAVFILES folder
Method B (Auto-restore): gF3(SAVEAUT)
→ Creates backup in SAVFILES folder that auto-loads after restart/update
Method C (Named backup): SAVEST
→ Creates named backup in STATE folder (e.g., MYNAME, BU001)

Step 2: Backup FAT Disk Contents to Computer

Activate USB Disk: f1, F2(ActUSB) OR f[1] PREF, DMCP, then item 6
On your computer, copy ALL folders from the R47 drive, including:

SAVFILES folder (contains your SAVE file)
SAVFILES folder (contains SAVEAUT file)
STATE folder (if using SAVEST)
All your work files

Important: Eject/safely remove the drive on your computer FIRST
Only after the OS releases the drive, disconnect the USB cable



----------------------------------------
Not part of the minimum update procedure
----------------------------------------
Sometimes you may want to clear the complete disk after you have all your needed backup copies on PC. Format it like this:
  OPTIONAL: FORMAT FAT DISK (After backing up to PC)
  When to do this: Only if you need to completely clear the disk

  7. System Menu
  5. Format
  2. Format FAT Disk
  EXIT
  4. Reset to DMCP menu
  6. Activate USB Disk
  Observe the disk name is now DMCP
  You can change this from your PC/Mac: Just rename from your file manager.

At this point you may want to re-instate the STATES and the SAVFILES folders and copy your previously save states back.
  1. After the disk format, or firmware update has been completed and your calculator is running, retrieve the SAVE file or SAVEST file from your FAT drive into your calculator.
  2. From the backed up SAVEFILES folder and/or from the STATE folder on your computer, store all files in your backup in the ame foldders oin the R47 FAT drive.
  3. Go back into the I/O menu of the R47 and select either
  3a. LOAD your backup, or
  3b. LOADST and select the SAVEST file that was created above.



----------------------------------------
FIRMWARE UPDATE PROCEDURES
----------------------------------------

Option 1: Update R47.pg5 File Only
----------------------------------------
f[1] PREF, DMCP, then item 6
With USB activated, transfer R47.pg5 to the root of the flash drive
Safely Remove Hardware and Eject Media (Eject R47 USB Device)
Press item 3 (Load program)
Navigate with arrows to R47.pg5
Press Enter to select
Press Enter to continue
Press any key to start
Press EXIT to continue
Restore your state: g0, fF3 (Skip if you used SAVEAUT)
Verify: gEXIT(INFO) shows version information


Option 2: Update DMCP5+R47 Combined Firmware (.bin file)
--------------------------------------------------------
f[1] PREF, DMCP, then item 6
With USB activated, transfer DMCP5_flash_X.XX_R47-00.109.02.XXXX.bin to root directory

One-time action: Rename backup files in SAVFILES folder if this is your first update since version 00.109.02.07b13.1.
  (This is not needed if you already did this after the 00.109.02.00b0 update)
  If you previously used SAVE: Rename C47.sav → R47.sav
  If you previously used SAVEAUT: Rename C47auto.sav → R47auto.sav

Safely Remove Hardware and Eject Media (Eject R47 USB Device)
Press [ENTER] to continue anyway
Press EXIT key to continue
Restore your state: g0, fF3 (Skip if you used SAVEAUT)
Verify: gEXIT(INFO) shows version information
